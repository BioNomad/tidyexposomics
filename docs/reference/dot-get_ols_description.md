# Internal - fetch OLS description for a term

Internal - fetch OLS description for a term

## Usage

``` r
.get_ols_description(ontology_id, ontology_prefix)
```

## Arguments

- ontology_id:

  Term ID (e.g., `"HP:0004322"`).

- ontology_prefix:

  Prefix (e.g., `"HP"`).

## Value

A character scalar description or `NA_character_`.
