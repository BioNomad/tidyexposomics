# Internal - load ontologies

Internal - load ontologies

## Usage

``` r
.load_ontologies(use_demo = TRUE, ...)
```

## Arguments

- use_demo:

  Logical; if TRUE, load packaged demo objects `hpo`, `ecto`, `chebi`.

- ...:

  Optional named overrides: `hpo`, `ecto`, `chebi`.

## Value

A list with elements `hpo`, `ecto`, `chebi`.
